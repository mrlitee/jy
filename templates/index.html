Real-time architecture

      
    
        
  • Backend menjalankan satu I/O worker thread per session — semua perintah ECU diserialisasi di sana.
    • 
        
  • Sampler latar terus membaca PID dengan jadwal adaptif: RPM/TPS @ 10 Hz, suhu/baro @ 0.2–1 Hz.
    • 
        
  • Browser berlangganan via Server-Sent Events di /api/stream — tidak ada polling.
    • 
        
  • Klik DTC scan / actuator test akan mendahului sampling (priority queue), sehingga respons terasa instan.
    • 
      


      
Tips

      
    
        
  • Kunci kontak motor harus ON, tegangan aki ≥ 11V
    • 
        
  • Beberapa ELM327 clone bermasalah pada protokol KWP. Kalau gagal, coba brand Generic OBD-II
    • 
        
  • Halaman bisa diakses dari HP lain di WiFi yang sama: http://<ip-termux>:5000
